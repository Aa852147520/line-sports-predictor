# LINE Sports Predictor Bot V2

## 功能
- LINE Webhook
- NBA 分析
- MLB 分析
- 足球分析
- 大小分方向
- 串關推薦
- VIP 指令入口
- 風險提醒

## Render 設定
Build Command:
```bash
npm install
```

Start Command:
```bash
npm start
```

Environment Variables:
```txt
LINE_CHANNEL_ACCESS_TOKEN=你的 Token
LINE_CHANNEL_SECRET=你的 Secret
```

Webhook:
```txt
https://你的Render網址.onrender.com/webhook
```
